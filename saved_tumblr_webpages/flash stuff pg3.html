    vertical motion graphic generator

    .jsfl file here

    for now this is a rough prototype but works. it’s not supposed to do much anyway. basically this creates a graphic containing vertical eased motion for curved motion already. this is supposed to be used with a nested symbols/tweens technique.

    it always needs adjusting afterwards but if you’re really lazy to create a graphic with this rough stuff, this may be useful. i don’t know.

    leave the symbol name prompt blank so that it generates an unique symbol name for you (“Symbol n”, where n is current symbol count of your document + 1).

    random selector

    .jsfl file here

    creates a movieclip, which you can populate with content and use it in order to have elements selected randomly (with actionscript) in your animation. i mostly use it for random props that i need to display, whenever i don’t feel like controlling those by hand.

    there’s an alternate version where you can instead select content on the library and it will populate the movieclip with whatever you selected.

    frame selection counter (+ useless layer counter thing)

    .jsfl file is here

    counts how many frames are in your selection. kinda handy when you’re trying to sync things. just a little convenient info box.

    there’s a layer counter too that will also tell you how many frames you have in the current scene. this one is even more useless :D

    image compression type changer

    .jsfl script is here

    sets the compression type of selected images in the library to lossless.

    a slightly more advanced script is “Library Bitmap Editor” by Devon O. Wolfgang. you can set the bitmaps to jpeg and also turn on smoothing for them.
