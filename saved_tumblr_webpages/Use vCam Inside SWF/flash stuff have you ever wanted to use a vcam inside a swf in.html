    have you ever wanted to use a vcam inside a swf in after effects? well, look no further! this collection of scripts is here to make that a (mostly) possible reality

    you can download everything you need + an example fla/xml here

    there are a few gotchas, KEEP THESE IN MIND BEFORE ASKING ANYTHING:

    - the version of flash you’re using must be higher than CS3 (currently only tested on CS3)

    - usage generally goes like this:

    first select all the frames of the layer you want to export. this should be your vcam layer. once you have all the frames selected, click on the “Export Motion XML for AE” command. it will then ask you to save a file somewhere. give the file a name with .xml at the end.

    once that is done, go in after effects, then click on File > Scripts > vcam motion importer.jsx. it will ask you for the xml file you saved. find it, then click ok and it will start parsing the xml and setting up everything. this may take a while, and if you exported a huge sequence, AE may appear to freeze. just let it run, eventually it will finish.

    once that’s all done, you’ll end up with a camera (flashCamera) and a solid (camPlaceholder). the solid will be invisible, and the camera will already have expressions to link it to the solid.

    throw your swf in the composition. you may need to reposition it, but that’s it!

    - if your vcam movement contains tweens anywhere, make these fbf sequences. the reason for this is that tweens are parsed, but their easing isn’t, making them pretty much useless.

    generally just make everything a huge fbf sequence. you can do this ONCE you’re done doing the vcam movement in your swf.

    - put both jsfl files on flash’s command folder. for flash CS3 it should be at:

    %LOCALAPPDATA%\Adobe\Flash CS3\en\Configuration\Commands\

    higher flash versions probably follow the same pattern. no idea where this happens to be in a mac. figure it out.

    the jsx file goes on after effects’ script folder. in AE CS4 it is at:

    %PROGRAMFILES%\Adobe\Adobe After Effects CS4\Support Files\Scripts

    - in the first frame of your fla, the vcam should be unchanged; as in, in the first frame JUST DRAG AND DROP IT AND LEAVE IT ON THE CENTER OF THE STAGE AS IT IS. if you do anything else it will screw up the placement of everything.

    anything else i didn’t mention here, feel free to send an ask about.

    many thanks to all those who did the initial testing!
